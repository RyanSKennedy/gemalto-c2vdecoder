Sentinel® LDK License Generation API - C# Sample for decode C2V from cloned Sentinel SL keys and compare System and Reference fingerprint (NON OFFICIAL): Readme

Overview

This sample demonstrates how to use of Sentinel LDK License Generation API for decoding C2V in the C# language.

Sample provide:

  • Possability to create report in *.xlsx format.
  • Can be using with multiple batch codes without re-configuration and re-building of the sample.
  • Sample can showing blocked products, mismatch criteria's in fingerprint and some other data from decoded C2V.
    Like:
       - update counter;
       - haspvlib version;
       - hasp fridge version;
       - clone protection scheme's for products.

Description of the Sample Flow

The sample consists of several functions. The default flow is as follows:

  1. Initialize Licgen API session

  2. Get Licgen API version and put it to title of the app

  3. Next you can chouse your Batch code which you prefere to use from ListBox.
    Remeber: before that, *.hvc file with your Batch code should be put in to directory: "...\*C2V_Decoder_base_dir*\VendorCode\"

  4. After selection of Batch code will be avaliable button Browse for choose your C2V file from cloned Sentinel SL key. Choose your C2V file.

  5. After selection of your C2V file will be available button Start for start decoding and analize C2V, push it. (Here working Licgen API function for decode current state.)

  6. If C2V was corrected, you got the result in DataGridView and in TextBox with results + will be available button Save as - for saving report as *.xlsx file.

  7. If something happen whrong you got error via MessageBox or in TextBox with results.
